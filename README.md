# webpart-spfx

## Summary

A React TypeScript web component built using SharePoint Framework (SPFx). This carousel displays tutorial videos with filtering options for:
- Pillar: Quantum, Harmony, CloudGuard, Infinity
- Level: All Levels, Advanced, Fundamental, Expert
  
Each card represents an object, displaying its details. 

Technical Highlights:
- Data passed as parameters to SPFx.
- Styled with TailwindCSS.
- Fully integrated with SharePoint data.

## Features

Description of the extension that expands upon high-level summary above.

This extension illustrates the following concepts:

- Dynamic Filtering: Enables filtering of tutorial videos by multiple parameters, such as pillar and skill level.
- Data Integration: Seamlessly integrates JSON data from SharePoint and renders it dynamically in the carousel.
- User Interaction: Provides an "Edit Properties" feature for users to view and edit the full JSON object tied to each video.
